## Usage

Mouse:  
* Left mouse button:  
  Draw lines/boundaries
* Right mouse button:  
  Set/move light source
* Mouse wheel:  
  Increase/decrease number of rays 

Keys:  
* Key b:  
  Add/Remove invisible lines/boundaries at the edge of the window
* Key h:
  Hide/reveal the drawn lines
* Key z:
  Delete last drawn line
